# Generate diff

Compares two configuration files and shows a difference.

## Setup

Clone repository
```sh
git clone git@github.com:SaenkoJr/backend-project-lvl2.git
```

Install deps
```sh
make install
```
or
```sh
npm install diff-saenkomm
```
